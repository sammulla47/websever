WhatsApp Float Button

A lightweight and flexible WordPress plugin to add a floating WhatsApp chat button with app & web support, custom positioning, icons, and mobile visibility control.

‚öô Installation

  1. Download the plugin ZIP file.
  2. Upload whatsapp-float-button to /wp-content/plugins/.

  4. Go to WhatsApp Float in the admin menu.
  5. Configure phone number, message, icon, and position.
